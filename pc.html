Under the cpu cooler you can find the CPU, also called a central processor, main processor or just processor, is the electronic circuitry that executes instructions comprising a computer program. The CPU performs basic arithmetic, logic, controlling, and input/output (I/O) operations specified by the instructions in the program.

Graphics processing unit, a specialized processor originally designed to accelerate graphics rendering. GPUs can process many pieces of data simultaneously, making them useful for machine learning, video editing, and gaming applications.

A power supply is a component that supplies power to the pc

Random access memory is a computer's short-term memory, which it uses to handle all active tasks and apps.

The motherboard is the part of the computer that connects every part of the computer together.
